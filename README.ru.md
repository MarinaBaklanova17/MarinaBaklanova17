## Избранные проекты

### Читаемость сказок, сгенерированных LLM — сравнительный анализ

Магистерское исследование (2025). Сравнительный анализ текстов, произведённых 19 большими языковыми моделями, против 16 классических сказок. Инструменты: `textstat`, `statsmodels`.

- **[Story-generation-process](https://github.com/MarinaBaklanova17/Story-generation-process)** — пайплайн генерации на 19 LLM × 16 исходных сказок
- **[LLM_readability_score_analysis](https://github.com/MarinaBaklanova17/LLM_readability_score_analysis)** — статистическая оценка (ANOVA, Tukey HSD) метрик читаемости

*Ключевой вывод:* ни одна модель не воспроизводит читаемость оригинальных сказок полностью; отдельные LLM превосходят других в конкретных измерениях читаемости.

### Real-Time Helmet Detection via Deep Learning

Детекция касок на промышленных видеозаписях с помощью YOLOv8 и Faster R-CNN.
Стек: PyTorch, TensorFlow, OpenCV. *Код предоставляется по запросу.*
